# pf-shop
React 기반의 반응형 쇼핑몰 UI로, 접근성과 성능 개선을 중심으로 제작.

## 🎯 목표
- 접근성 있는 UI 구현 (ARIA, 포커스트랩, 키보드 조작)
- 유지보수 가능한 구조화 (컴포넌트, SCSS, 데이터 분리)

## 🛠 기술 스택
React, Vite, SCSS, JSON, Git

## 📘 주요 기능
- 상품 리스트 / 필터 / 상세보기 모달
- 키보드 탭 / ESC / 포커스트랩 완비
- 반응형 (1440 / 1024 / 768)

## 💬 역할
- UI 구조 설계, 접근성 개선, 퍼포먼스 최적화, 문서화
